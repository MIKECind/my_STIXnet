SID-History attribute Article 12/14/2020 3 contributors Feedback In this article Contains previous SIDs used for the object if the object was moved from another domain. Whenever an object is moved from one domain to another, a new SID is created and that new SID becomes the objectSID. The previous SID is added to the sIDHistory property. Entry Value CN SID-History Ldap-Display-Name sIDHistory Size - Update Privilege This value is set by the system. Update Frequency Each time the object is moved to a new domain.
